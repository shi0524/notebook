## 如何用对列实现栈结构, 或如何用栈实现队列结构

```
1）如何用栈结构实现队列结构
思路:
    准备两个栈,
    入栈总是加入到stack1
    出栈:
        如果stack2 为空, 将 stack1 所有的数据依次弹出, 压入stack2
        如果stack2 不为空的话,直接从 stack2 弹出
    注意:
        压栈只进 stack1
        出栈只从 stack2 出
        只有stack2 数据为空的时候 stack1 才能往 stack2 倒 数据
        stack1 往 stack2 倒数据, 必须一次性倒空

2）如何用队列结构实现栈结构
```
