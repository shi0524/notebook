

## 线段覆盖点问题(用窗口实现)

```
给定一个有序数组arr，代表坐落在X轴上的点
给定一个正数K，代表绳子的长度
返回绳子最多压中几个点？
即使绳子边缘处盖住点也算盖住

思路:
    小贪心：
        绳子的某一端必定盖住一个点，最优
    假设：绳子长度为N
    二分法：
        绳子的右端盖住某一个点 i，用二分求第一个 0 ~ i 之间 >= i-N 的数的索引 j
        ans = i - j + 1
    窗口：
        三个变量 ans, L, R

        总是以 L 盖住某个点 L
        R 向右移动，移动到 R - L > N
        收集答案：ans = max(ans, R - L)

        L 向左移动
        R 向右移动，移动到 R - L > N
        收集答案：ans = max(ans, R - L)
```
