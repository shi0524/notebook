
# 双指针

## 两两组队比赛

```
给定一个数组arr，代表每个人的能力值。再给定一个非负数k
如果两个人能力差值正好为k，那么可以凑在一起比赛
一局比赛只有两个人
返回最多可以同时有多少场比赛

思路:
    小贪心：排序, 如果多个符合条件，选取最小的两个
    双指针：L, R
    申请一个记录 已经比过赛的 集合   或者 一个等长 bool 数组
    L, R 同时从 0 号位开始
    L, R 的差值小于 K, R 向右移动
    L, R 的差值等于 K, 匹配成功一组, 将R位置放入集合, L、R 同时向右移动
    L, R 的差值大于 K, L 向右移动
        
讲解:
    大厂刷题班: 第3节 第4题 00:27:56
```
