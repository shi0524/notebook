
# 贪心算法

## 两种字符G B 交换

```
一个数组中只有两种字符'G'和'B’，
想让所有的G都放在左侧，所有的B都放在右侧
但是只能在相邻字符之间进行交换操作，
返回至少需要交换几次

思路:
    小贪心：
        同样的字符，顺序不变换
        第1个 G 放在 0 位置
        第2个 G 放在 1 位置
        第i个 G 放在 i-1 位置

    伪代码：
        ans = 0
        index = 0
        for i in str:
            if str[i] == 'G':
               ans += i - index
               index += 1 
讲解:
    大厂刷题班: 第1节 第4题 00:41:55
```
